⚙️ Configuración de Google Sheets

Paso 1: ID de la Hoja (para LEER datos)

Paso 2: URL del Script (para ESCRIBIR datos)

📋 Instrucciones de configuración:

  1. Crear Google Sheet: Andá a sheets.google.com y creá una hoja nueva
  2. Encabezados: En la Fila 1, pegá:fecha,lab_t,lab_f,lab_m,med_t,med_f,med_m,odonto_t,odonto_f,odonto_m,atletica_t,atletica_f,atletica_m,socio_turno_t,socio_turno_f,socio_turno_m,socio_proceso_t,socio_pend_t,socio_ausente_t,socio_total_t,socio_total_f,socio_total_m,admin_turno_t,admin_turno_f,admin_turno_m,admin_ausente_t,admin_ausente_f,admin_ausente_m,admin_total_t,admin_total_f,admin_total_m,doc_turno_t,doc_turno_f,doc_turno_m,doc_ausente_t,doc_ausente_f,doc_ausente_m,doc_total_t,doc_total_f,doc_total_m,proceso_t,proceso_f,proceso_m,potencial_t,potencial_f,potencial_m
  3. Publicar para leer: Archivo → Compartir → Publicar en la web → CSV → Publicar
  4. Copiar ID: De la URL copiá el ID (entre /d/ y /edit) y pegalo en "Paso 1"
  5. Crear Script: En el Sheet, andá a Extensiones → Apps Script
  6. Pegar código: Borrá todo y pegá este código:function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow(data); return ContentService.createTextOutput(JSON.stringify({status:'ok'})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { return ContentService.createTextOutput('Script activo'); }
  7. Implementar: Clic en Implementar → Nueva implementación
  8. Configurar: Tipo: "Aplicación web" → Acceso: "Cualquier persona" → Implementar
  9. Autorizar: Revisá permisos → Seleccioná tu cuenta → Avanzado → Ir a (nombre) → Permitir
  10. Copiar URL: Copiá la URL que termina en /exec y pegala en "Paso 2"
