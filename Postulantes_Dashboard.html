Usamos JSONBin (gratis) para guardar los datos en la nube. Seguí estos pasos:

1. Entrá a jsonbin.io y creá una cuenta gratuita
2. Andá a API Keys y copiá tu clave
3. Creá un nuevo Bin con este contenido: {"policia":[],"bomberos":[],"nextId":1}
4. Copiá el Bin ID (está en la URL)
5. Pegá ambos acá abajo y listo
